# Student Performance Analysis

## Overview
This project analyzes student performance data using Python.

## Tools Used
- Python
- NumPy
- Pandas
- Google Colab

## Dataset
The dataset includes:
- Hours Studied
- Previous Scores
- Extracurricular Activities
- Sleep Hours
- Sample Question Papers Practiced
- Performance Index
